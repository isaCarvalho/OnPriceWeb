Criar Conta

Para criar uma conta, o usuário, dono do estabelecimento, deve preencher o formulário com os dados solicitados.

Logo após, será redirecionado para realizar o login no On Price.

Entrar

Insira os dados do estabelecimento (nome e senha) e clique no botão Entrar abaixo do formulário.

O usuário será redirecionado para o espaço do mercado, onde poderá gerenciar seus produtos.

Criar Produtos

Preencha o formulário com os dados do produto. Lembre-se de preencher o campo de unidade como Kg, Pacote, etc.

O produto aparecerá no espaço do mercado.

Mercados

Clique em produtos na tela inicial.

O usuário será redirecionado para uma tela com uma lista dos mercados cadastrados no On Price.

Produtos

Na listagem de estabelecimentos descrita no passo anterior, clique no mercado desejado

O usuário será redirecionado para uma tela com a lista dos produtos deste estabelecimento cadastrados no On Price.
